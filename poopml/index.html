PoopML

PoopML is both a content and structure expression standard. It originates in work on the Feces Expression Content and Arrangement Language (FECAL).

A PoopML implementation should encourage well-formed representations whenever possible. However, due to the flexibility of PoopML, you may have provide a structured framework for parsing.

The primary element set, which follows below, is also referred to as "Dublin Corn".

Primary element set

  • <poop> - the root element
  • <size>
  • <shape>
  • <smell>
